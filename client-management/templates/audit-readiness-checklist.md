## 30 Days Before Audit

### Documentation
- [ ] All policies signed and dated by leadership
- [ ] Risk register populated with at least 10 risks
- [ ] AI system inventory complete
- [ ] Procedures documented for all key processes
- [ ] Scope statement clearly defined

### Evidence
- [ ] Training records for all staff (minimum 3 months)
- [ ] Meeting minutes from management reviews
- [ ] Internal audit completed and report finalized
- [ ] Corrective action log with closed items
- [ ] Vendor assessment documentation

### Logistics
- [ ] Auditor contact information confirmed
- [ ] Audit agenda received and reviewed
- [ ] Conference room/space reserved
- [ ] Key personnel availability confirmed
- [ ] NDA signed with certification body (if required)

## 7 Days Before Audit

- [ ] All evidence organized and accessible
- [ ] Document index created
- [ ] Key personnel briefed on audit process

- [ ] Staff aware of audit dates
- [ ] Interview schedule shared

## Common Auditor Questions

Be prepared to answer:

1. **"How do you identify AI risks?"**
   - [ ] Know the risk assessment process
   - [ ] Can explain top 3 risks and mitigations

2. **"How do you ensure data quality?"**
   - [ ] Explain data validation procedures
   - [ ] Show examples of data quality checks

3. **"How do you monitor AI performance?"**
   - [ ] Describe metrics tracked
   - [ ] Show evidence of monitoring

4. **"What happens when something goes wrong?"**
   - [ ] Explain incident response process
   - [ ] Show example incident (if any)

5. **"How do you manage third-party AI?"**
   - [ ] List vendors used
   - [ ] Explain vendor assessment process

## Post-Audit Actions

### Immediately After
- [ ] Thank auditor
- [ ] Note any verbal feedback
- [ ] Schedule debrief with team

### Within 24 Hours
- [ ] Document auditor observations
- [ ] Identify any gaps discovered
- [ ] Begin corrective actions (if needed)

### Within 1 Week
- [ ] Receive formal audit report
- [ ] Review findings with leadership
- [ ] Develop corrective action plan
- [ ] Submit evidence for any minor non-conformities
